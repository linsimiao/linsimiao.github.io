写个标题以防下面全灰了

  • maven的项目推荐最外层的文件不要有代码，parent的pom.xml里面加入<packaging>pom<packaging>。
  • 如果依赖的包用了nexus进行版本控制的话，在maven clean package中间加入-Pnexus。
  • 如果要忽略测试的话，加入-DskipTests=true，如果连测试的编译都想忽略的话，那么就换成-Dmaven.test.skip=true
  • 当项目没有如预期一样load下载好的package时，右键点击project，maven，reimport，就可以了。原因是代码环境发生了变化，但是机器脑子反应不过来。
